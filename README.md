ie# cybershield
a simple fastAPI-based network scanner with a user interface for security analysis
Features
- Scan local network for active devices
- User-friendly web interface
- Save scan results to a text file

Technologies Used
- Python
- FastAPI
- HTML/CSS

How to Run
1. Clone this repository
2. Run `interface.py`
3. Open the browser interface to start scanning
