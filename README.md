CyberShield* is a fast and user-friendly FastAPI-based network scanner designed to detect active devices on your local network and assist with security analysis.

Features

- Scans for active devices on the local network  
- User-friendly web interface  
- Saves scan results to a text file  

Technologies Used

- Python  
- FastAPI  
- HTML/CSS  

Installation and Usage

1. Clone the repository: git clone https://github.com/iremclk34/cybershield.git
2. Install the requirements:   pip install -r requirements.txt
3. Run the application:  python interface.py
4. Access the web interface in your browser to start scanning.
